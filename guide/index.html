# Introduction

Pine is a library for creating webservers. Our goal with Pine is it to minimize/eliminate the need of any dependency. By not depending on any third party and offloading heavy tasks like body parsing to only when the developer wants to, we were able to achieve a high performance standard.

WARNING

Please note that this project was written by one computer science student and even though it might look good. Do not use it for any serious project/production builds.
