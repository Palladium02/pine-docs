# First steps

To give you a first feel for Pine we will build a very simple Hello world example before we go into the depth of everything.

# Creating an Application instance

The Application class gives you an interface where all functionalities come together. So you will not come far without creating one. Simply import the Application class and create an instance.

import {Application} from "pine";

const app = new Application({
  port: 80,
});

Now that we have created an application we see that it is not doing anything. So why do not we add a couple route handler. For that we need the Router class, which brings us to the next topic.

# The Router

The Router gives you all the methods you need to create route handler for all common http methods.

import {Router} from "pine";

const router = new Router("");

router.get("/", async ({response}) => {
  response.body("Hello World!");
  response.send();
});

The snippet above creates a Router instance with a handler handling http get requests to the route: / and it responds with our all favorite line. What is actually going on in the handler will be covered in one of the following sections so do not worry.

# Linking things together

Now having created these two snippets we are almost there. The last step is to link the router and the application and make the server listen. For exactly that the Application provides you with two methods: use and listen.

import {Application, Router} from "pine";

const app = new Application({
  port: 80,
});
const router = new Router();

router.get("/", async ({response}) => {
  response.body("Hello World!");
  response.send();
});

app.use(router);

app.listen();

If we now go into a browser or any other http client and make a get request, assuming you are running this locally, to http://localhost we should be presented with a lovely Hello World!. From here things start to become very interesting so if your interest was sparked keep reading to get a full understanding of how things work with Pine.
