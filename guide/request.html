# Request

Represents the request and is created for every request.

# header(key?: string)

The header method is a getter method for the request headers. It might receive an optional key parameter. If not provided header will return all request headers else only a specific header is returned.

# body

Returns the request body. Further reading.

# params(key?: string)

The params method is a getter method for the route params. It might receive an optional key parameter. If not provided params will return all request route parameter else only a specific parameter is returned.

# query(key?: string)

The query method is a getter method for the query parameters. It might receive an optional key parameter. If not provided query will return all query parameters else only a specific parameter is returned.

# cookies(key?: string)

WARNING

Not implemented.

# url

Returns the request url.
